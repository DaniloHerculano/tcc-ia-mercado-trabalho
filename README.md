# TCC - IA e Mercado de Trabalho Brasileiro

Projeto de Ciência de Dados voltado para classificação do risco de automação de ocupações brasileiras utilizando técnicas de Machine Learning.

## Tecnologias

- Python
- Pandas
- Scikit-learn
- Random Forest
- Matplotlib
- Seaborn

## Estrutura

- Pré-processamento de dados
- Análise exploratória
- Treinamento do modelo
- Avaliação do modelo

## Execução

```bash
pip install -r requirements.txt
python src/main.py
